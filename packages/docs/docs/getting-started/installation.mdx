# 설치

Edim 을 사용하기 위해서는 먼저 ProseMirror 를 설치해야 합니다.

```bash npm2yarn
npm install prosemirror-state \
            prosemirror-model \
            prosemirror-view
```

Edim 은 여러개의 패키지로 나뉘어 배포됩니다.
 
```bash npm2yarn
npm install @edim/core \
            @edim/paragraph \
            @edim/heading \
            @edim/flat-list \
            @edim/menubar
```
