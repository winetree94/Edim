# 사용자화

**Edim**은 에디터의 각 구성 요소를 개별로 설치하여 사용자가 원하는 에디터의 기능만 사용하거나, 자신만의 기능을 추가할 수 있습니다. Edim 의 각 패키지는 ProseMirror 에서 사용할 수 있는 서식(Node, Mark)과 플러그인(Plugin)으로 구성됩니다. 아래는  `@edim-editor/core` 패키지를 통해 최소한의 기능으로 에디터를 생성하는 코드 예시입니다.

```typescript
import { Plugin } from 'prosemirror-state';
import { Schema } from 'prosemirror-model';
import { EditorView } from 'prosemirror-view';
import {
  EDIM_DOC_NODES,
  edimCorePlugins,
} from '@edim-editor/core';
import {
  EDIM_PARAGRAPH_NODE,
} from '@edim-editor/paragraph';

const view = new EditorView(document.querySelector('#editor'), {
  state: EditorState.create({
    schema: new Schema({
      nodes: {
        ...EDIM_DOC_NODES,
        ...EDIM_PARAGRAPH_NODE,
      },
    }),
    plugins: [
      ...edimCorePlugins()
    ],
  }),
});
```
