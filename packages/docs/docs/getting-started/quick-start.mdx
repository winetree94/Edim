# 빠른 시작 가이드

**Edim**의 각 구성요소는 여러개의 패키지로 나뉘어 배포됩니다. 사용자는 필요한 구성요소만을 선택하여 자신만의 에디터를 구성하거나, 모든 구성 요소가 통합된 `@edim-editor/preset` 패키지만을 사용할 수도 있습니다. 본문은 해당 패키지를 사용하여 최소한의 코드로 에디터를 생성하는 방법을 설명합니다.

## 의존성 설치

Edim 을 사용하기 위해서는 기반인 ProseMirror 를 함께 설치해야 합니다. 사용하는 패키지 매니저에 따라 아래의 설치 명령어를 사용합니다.

```bash npm2yarn
npm install prosemirror-state \
            prosemirror-model \
            prosemirror-view \
            @edim-editor/preset
```

## 에디터 생성

```html title="/src/index.html"
<html>
  <head>
    <link rel="stylesheet" href="/styles/global.css">
  </head>
  <body>
    <div id="editor"></div>
  </body>
</html>
```

```scss title="/styles/global.scss"
@import '@edim-editor/preset/styles/preset.scss';
```

:::info

scss 를 선호하는 경우 `@edim-editor/preset/styles/preset.scss` 를 사용할 수도 있습니다.

:::

```typescript title="/src/index.ts"
import { EditorState } from 'prosemirror-state';
import { EditorView } from 'prosemirror-view';
import { EDIM_PRESET_SCHEMA, edimPresetPlugins } from '@edim-editor/preset';

const view = new EditorView(
  // 에디터가 생성될 위치의 Div Element 를 지정해야 합니다.
  document.querySelector('#editor'),
  {
    state: EditorState.create({
      schema: EDIM_PRESET_SCHEMA,
      plugins: edimPresetPlugins(),
    }),
  },
);
```
